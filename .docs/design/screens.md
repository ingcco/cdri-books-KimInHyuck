## 렌더 검수 결과 (PNG 실측, 2026-07-08 — `~/Downloads/certicos Books/*.png`)

> API rate-limit로 프레임별 렌더 PNG를 직접 대조. 이게 최종 SOT.

### ✅ 일치 (수정 불필요)
- h1 "도서 검색" 타이틀 **존재** / 검색박스(pill·회색·돋보기·상세검색 outline 버튼) / count "총 N건" 숫자 primary
- collapsed·expanded 아이템 레이아웃 / **할인가 없으면 미노출** / 빈상태(teal 책 일러스트 + "검색된 결과가 없습니다." 중앙정렬)
- 무한스크롤·10개/페이지·구매하기 새탭·하트 토글·**상호배타**(전체↔상세, key 리셋)
- 드롭다운 옵션: 제목/저자명/출판사 (title/person/publisher)

### ❌ 수정 필요
1. **⑥ 헤더** — Figma: 로고 **페이지 좌측(~150px, 960 컬럼 밖)** + 탭 **중앙 정렬**, 우측 빈 공간. 내 `max-w-960 mx-auto justify-between`(탭을 우측 끝에)은 틀림
2. **상세검색 팝오버**(`pop.png`) — 드롭다운 "제목 ▼"이 **박스 아닌 borderless 텍스트+chevron**, input underline과 baseline 정렬(=라인 어긋남 원인). 내 Dropdown은 `w-100` 박스. + **× 닫기 버튼 누락**(top-right)
3. **count→리스트 간격** — Figma ~36px, 내 `result` `gap-4`(16px)
4. **본문 좌우 인셋** — Figma 리스트/구분선 960 풀폭, 내 컨테이너 `px-4`로 928(16px 인셋)

### ⑤ 히스토리 (디자인 미명시 → 지원자 판단)
4프레임·pop 어디에도 검색 히스토리 드롭다운 디자인 없음(유의사항 "미명시는 판단"). 내 `historyList` 스타일 유지 + **동작만 추가**: Esc 닫기·방향키 네비·hover/focus 하이라이트.
